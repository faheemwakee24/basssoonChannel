## 💡 Quick Examples

### Success Toast

```typescript
dispatch(
  showSnackbar({
    message: 'Success!',
    type: 'success',
  }),
);
```

See: **TOAST_SETUP_GUIDE.md** → Usage Examples

### Error Toast

```typescript
dispatch(
  showSnackbar({
    message: 'Error!',
    type: 'error',
  }),
);
```

See: **TOAST_QUICK_REFERENCE.md** → 4 Types

### With Try/Catch

```typescript
try {
  await doSomething();
} catch (error) {
  dispatch(
    showSnackbar({
      message: error.message,
      type: 'error',
    }),
  );
}
```

See: **TOAST_SETUP_GUIDE.md** → Advanced Usage
